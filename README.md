# Readiness Prototype

## Da smartphone (QR)
Inquadra il QR code → si apre direttamente sul browser del telefono.

![QR](qr.png)

URL diretto: https://ragno236.github.io/provafinale/

## Da PC (locale)
Scarica questa cartella, poi da terminale:

```
node serve.js
```

Apri il browser su **http://localhost:8765**

> Requisito: [Node.js](https://nodejs.org) installato (versione 14+)
